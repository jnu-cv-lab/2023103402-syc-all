# 计算机视觉实验一：Python视觉开发环境搭建与OpenCV图像基本读写

## 实验目的
1. 搭建Python+OpenCV图像处理环境
2. 掌握图像读取、信息输出、格式转换、保存等基本操作
3. 学习使用NumPy对图像进行简单像素操作

## 实验环境
1. 系统：WSL Ubuntu
2. 语言：Python 3
3. 库：OpenCV、NumPy、Matplotlib

## 实验环境
1. 使用OpenCV读取彩色图像
2. 输出图像宽度、高度、通道数、数据类型
3. 将彩色图像转为灰度图像
4. 保存灰度图与裁剪区域图像
5. 使用NumPy访问指定像素并完成区域裁剪

## 运行方式
bash
python ex.py

## 实验结论
实验一进行了计算机环境的搭建，将opencv写入环境变量中。通过项目熟悉了vscode、ubuntu、python的使用和运行。完成了图像读取、显示、变换与保存操作。
